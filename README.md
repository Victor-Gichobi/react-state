
# React Class Component Checkpoint

## Description

This project demonstrates the use of React class components, state, event handling, conditional rendering, and lifecycle methods.

## Features

- Class-based component
- State management
- Toggle profile visibility
- Conditional rendering
- Lifecycle methods
- Timer showing how long the component has been mounted

## Technologies Used

- React
- JavaScript
- CSS
